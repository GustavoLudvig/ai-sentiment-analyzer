# AI Sentiment Analyzer 🥖
A professional sentiment analysis tool for bakery customer feedback, built with Python and Llama 3 (via Groq Cloud).

## 🚀 Features
- **AI-Powered**: Uses LLMs to understand context, not just keywords.
- **Secure**: Implements `.env` for API key protection.
- **Fast**: Integrated with Groq for near-instant responses.

## 🛠️ Technologies
- Python 3.x
- Groq Cloud API (Llama 3 Model)
- python-dotenv

## 📦 How to run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your `GROQ_API_KEY`
4. Run: `python main.py`
